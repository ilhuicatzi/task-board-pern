### PERN TASKS

This is a simple task manager app built with the PERN stack (PostgreSQL, Express, React, Node.js).

### Features

- Sign in and sign up with email and password
- Create, read, update, and delete tasks

### Technologies

- PostgreSQL
- Express
- JSON Web Tokens
- React
- Node.js
- Tailwind CSS
- Axios
- Cors
- React Hook Form
- React Router Dom
- React Icons
- Vite
